About the Project

Youtube-Sync is a project that aims to sync up video streams to allow students at the University of Illinois to view prerecorded lectures together in real time. This project uses the Video.js API for video playback and WebSockets to send information to and from the web server. Built by a student at the University of Illinois SRTI Lab.
